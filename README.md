# A Markdown to HTML Convertor

## Currently supports

`#`, `##`, ... -> `<h1>`, `<h2>`, ...

`---` -> `<hr>`

Unordered/ordered list

`![]()`, `<img>` -> `<img>`

`[]()` -> `<a>`

Plain text

## How to compile

The project uses `CMake` as building tool.
